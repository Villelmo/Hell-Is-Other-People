The Concept

This project is partially a satire, partially a commentary on my disdain for "social media", and partially an exploration of my own difficulties with severe social anxiety.

The title comes from an existentialist play by Jean-Paul Sartre entitled No Exit.

The Backend

This project was developed using the Express web application framework for Node.js and is hosted on the Heroku cloud application platform. It is powered by the FourSquare API.

The Map

The map is generated using the Google Maps API with custom styling. Boundaries were chosen based on the WOEID of Manhattan and centered on ITP.

The Math

The locations given to avoid other users are calculated by generating a Voronoi diagram with user check-ins as seeds. The diagram is generated using a javascript library by Raymond Hill.
