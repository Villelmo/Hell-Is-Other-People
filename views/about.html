The Concept

This project is partially a satire, partially a commentary on my disdain for "social media", and partially an exploration of my own difficulties with severe social anxiety.

The title comes from an existentialist play by Jean-Paul Sartre entitled No Exit.

The Backend

This project was developed using the Express web application framework for Node.js and is hosted on the Heroku cloud application platform. It is powered by FourSquare's Real-Time API.

The Map

The map boundaries were chosen based on the WOEID of Manhattan. The image was generated using TileMill and freely availble data. The vector overlay is generated in-browser using svg.js.

The Math

The locations given to avoid other users are calculated by generating a Voronoi diagram with user check-ins as seeds. Since Voronoi nodes are points equidistant to three or more sites, they work as a straightfoward way to find new points optimially distanced from the original set.

I'm certain that mathematicians out there are cringing at that description, but it works for my needs. The Voronoi diagram is generated using a javascript library by Raymond Hill.
